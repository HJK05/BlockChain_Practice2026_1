# ChainBoard
## 블록체인 기반 책임형 익명 게시판

> 익명은 보호하고, 책임은 기록한다.

---

# 프로젝트 소개

ChainBoard는 블록체인 기술을 활용하여 기존 인터넷 익명 커뮤니티의 문제점을 개선하기 위한 책임형 익명 게시판 프로젝트입니다.

기존 익명 커뮤니티는 자유로운 의견 표현이 가능하다는 장점이 있지만, 동시에 악성 댓글, 명예훼손, 허위사실 유포, 불법 홍보 등 다양한 문제점이 존재합니다. 또한 관리자 중심의 신고 처리 구조는 편파 처리 및 기록 조작 가능성을 가진다는 한계가 있습니다.

본 프로젝트는 단순 인터넷 실명제가 아닌, 블록체인 기반 익명 ID와 투명한 신고·투표 시스템을 통해 익명성과 책임성을 동시에 달성하는 새로운 형태의 커뮤니티 모델을 제안합니다.

---

# 프로젝트 목표

## 책임형 익명제 구현

- 실명 공개 없이 블록체인 기반 익명 ID 사용
- 사용자 활동 이력 및 신고 기록 관리
- 개인정보 보호와 책임성의 균형 달성
- 심각한 범죄 상황에서만 법적 절차를 통한 신원 확인 가능

---

# 기존 시스템의 문제점

## 기존 익명 커뮤니티 문제

- VPN 및 IP 우회 가능
- 반복적인 악성 행위자 추적 어려움
- 익명성을 악용한 책임 회피
- 피해자 구제 및 법적 대응 한계

## 관리자 중심 구조의 한계

- 관리자 개인 판단 기반 제재
- 신고 기록 조작 및 삭제 가능성
- 투명하지 않은 처리 과정
- 편파 처리 논란 발생 가능

---

# 해결 방안

ChainBoard는 다음과 같은 블록체인 기반 구조를 통해 기존 문제를 개선합니다.

- 블록체인 기반 익명 사용자 ID
- 신고 및 수정 기록 영구 저장
- 커뮤니티 투표 기반 판정 시스템
- 신고 누적 이력 추적
- 위변조 불가능한 기록 관리

---

# 주요 기능

## 1. 블록체인 기반 익명 ID

- MetaMask 지갑 기반 로그인
- 실명 공개 없는 사용자 식별
- 동일 사용자의 활동 기록 추적 가능

---

## 2. 게시글 관리 시스템

- 게시글 작성
- 게시글 수정
- 게시글 삭제 기록 저장
- contentHash 생성 및 기록

예시:

```text
contentHash:
7710db949d490f6e99254c81fd6dd30b37b89b4a8f811b98f38d7f093abc1963
```

---

## 3. 댓글 시스템

- 게시글별 댓글 작성 가능
- 댓글 단위 신고 가능
- 댓글 활동 이력 관리

---

## 4. 신고 시스템

### 신고 사유

- 불법 홍보
- 악의적 글
- 명예훼손
- 기타

기타 선택 시 직접 신고 사유 입력 가능

### 신고 기능

- 신고 시간 기록
- 신고 사유 저장
- 신고 내역 블록체인 기록
- 위변조 방지

---

## 5. 커뮤니티 투표 시스템

신고 접수 후 커뮤니티 사용자들이:

- 잘못 있음
- 잘못 없음

으로 투표를 진행합니다.

### 특징

- 관리자 독단 방지
- 다수결 기반 민주적 판정
- 투표 결과 기록 저장
- 투표 과정 투명성 확보

---

## 6. 반복 신고 이력 추적

동일 사용자가 특정 사유로 기존에 몇 번 신고되었는지 확인 가능합니다.

예시:

```text
불법 홍보 신고 누적 횟수: 4회
명예훼손 신고 누적 횟수: 2회
```

---

## 7. 단계별 제재 시스템

- 경고 시스템
- 누적 신고 기반 추적
- 반복 위반 사용자 제재
- 심각한 경우 법적 절차 연계

---

```text
↓
누적 신고 이력 관리
```

# 프로젝트 의의

본 프로젝트는 단순 ERC20 토큰 발행이 아닌:

- 블록체인의 불변성
- 탈중앙화 구조
